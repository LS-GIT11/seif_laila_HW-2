Overview

What the Counter class does

The Counter class is used to create a working counter on the page. It lets the user increase, decrease, and reset the number.


            export class Counter {
                constructor(selector, initialValue = 0) {
                    this.count = initialValue;
                    this.selector = selector;
                    this.mount();
                }
            }

This class is meant to be simple and reusable, so it can be used for different counters in the same project.

What problem it solves

The Counter class helps organize counter functionality into one place instead of rewriting the same code multiple times.

This makes the code easier to manage, easier to reuse, and easier to update later.

How To Use

Example HTML container markup

Here is an example of the HTML container markup for the counter:


                <div id="counter-container"></div>

This code creates a div element with the id "counter-container". This is where the counter will be rendered on the page.

Example instantiation code

Here is an example of how to instantiate the counter:


                const counter = new Counter('#counter-container', 0);

Example of parameters

The use of parameters is so we can customize the counter's initial value and the HTML container it will be rendered in. In other words, we can specify the selector for the container and the initial value for the counter.


                const counter = new Counter(selector, initialValue);

This above code demonstrates how to use the parameters to customize the counter's initial value and the HTML container it will be rendered in. Starting with "const counter", we create a new instance of the Counter class, passing in the selector for the container and the initial value for the counter.

Live Demo

Different initial values

To demonstrate different initial values, we can create multiple counters with different starting points. This is by creating a new counter (giving it a different variable name for each one) then adding the corresponding HTML container for each counter. Separating with a comma, we can add what number the counter should start from.

Shown below are counters with initial values of 0, 2, and 5.


                

                const counter1 = new Counter('#counter-container1', 0);
                

                const counter2 = new Counter('#counter-container2', 2);
                

                const counter3 = new Counter('#counter-container3', 5);

Step Counters

What if we also want the counter to increment by a specific step value?
