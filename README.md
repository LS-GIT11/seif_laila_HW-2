# Counter OOP JS 🌟

A small demo project using Object-Oriented Programming in JavaScript to create reusable counter components. Includes multiple counters with different starting values and a step counter variation.

## Installation 💻
1. Clone or download the repo  
2. Open `index.html` in your browser  
3. (Optional) Compile SCSS with Sass if you want to edit styles  

## Usage 🔢

- Click **Start** to reveal the counters  
- Use the **+ / - / Reset** buttons to interact with each counter  
- View different counters with different starting values  
- The step counter increases by a set value instead of 1
